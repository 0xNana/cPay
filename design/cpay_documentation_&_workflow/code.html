Smart Account (SA)

cPay utilizes modular Account Abstraction (ERC-4337) to handle gas-less execution and multi-signature corporate approvals.

Multi-Sig Support Gas Manager Time-Locks

ERC-7984 Implementation

// Initialize ISO batch execution
function executePayrollBatch(
    bytes32 batchIdentifier,
    PayrollInstruction[] calldata instructions,
    bytes calldata corporateSignature
) external onlyAuthorized {
    // 1. Verify pain.001 derived signature
    _verifyCorporateAuth(corporateSignature);

    // 2. Execute multi-call via Smart Account
    _processBatch(instructions);

    // 3. Emit event for pain.002 generation
    emit BatchStatusReport(batchIdentifier, "ACCP");
}
